Malaria Detection

This projct aims at identifying malaria parasitized cells in an infected person. To train the model, a coloured image dataset of numerous cells was collected which was then converted to black and white. Following this, through contour detection 5 areas are calculated. This data is then fed into RamdomForestClassifier which returns our desired data. Now, if a cell image is input in the model, we can find out whether the person is infected by malaria or not.

Image Classification

This is a multiclass image classifier which predicts the proecssed image from 10 predefined classes of the CIFAR-10 dataset. The model is based on CNN architecture with an accuracy of over 76% in 10 epochs, which improved on increasing the number of epochs to 25.
In this project, a convolution neural network in Keras with python on a CIFAR-10 dataset is built. First,the dataset is explored, and then the neural network is trained using python and Keras. The CIFAR-10 dataset consists of 60000 32x32 colour images in 10 classes, with 6000 images per class. There are 50000 training images and 10000 test images.
